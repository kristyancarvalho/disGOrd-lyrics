# DisGOrd Lyrics v0.1.0

First public binary release of DisGOrd Lyrics.

## Highlights

- Go implementation with no Python runtime dependency
- Linux and Windows release artifacts
- TOML-based configuration
- Discord custom status update loop
- Synchronized lyrics parsing
- Linux startup documentation
- Windows startup documentation

## Downloads

- Linux amd64: `disgord-lyrics-v0.1.0-linux-amd64.tar.gz`
- Linux arm64: `disgord-lyrics-v0.1.0-linux-arm64.tar.gz`
- Windows amd64: `disgord-lyrics-v0.1.0-windows-amd64.zip`
- Checksums: `checksums.txt`

## Important Notice

This application updates Discord custom status using user-provided credentials. Read the README before use and understand the risks before configuring your token.
